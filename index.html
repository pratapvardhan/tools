A collection of web apps and tools. Made for personal use. Some built with LLM assistance.

Tools

  • CSV Merge - merge multiple CSV files.
  • PDF Splitter - split PDF files into multiple parts.
  • Timezones - track time in multiple timezones.
  • Location Mapper - geocode locations and plot on map.
  • Image Color Quantizer - reduce number of colors in an image by combining them to nearest ones.
  • Notepad - using local storage, no tracking.
  • Notes in Browser tab - write notes in your browser.
